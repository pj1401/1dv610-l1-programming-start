# L1. Börja programmera igen

## Vad ska applikationen göra?

- Användaren ska mata in ett namn och få en hälsning tillbaka.

För att starta applikationen:

```bash
npm start NAME
```

Exempel utskrift:

```bash
npm start Patrizio
```
